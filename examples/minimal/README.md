# Minimal Example

Basic tool management without presets. Defines resources using raw CUE definitions.

## Directory Structure

```
minimal/
├── cue.mod/module.cue    # CUE module (generated by tomei cue init)
├── tomei_platform.cue    # Platform @tag() declarations (generated)
├── tools.cue             # CLI tools via aqua registry
├── runtime.cue           # Go runtime + tools via go install
└── rust.cue              # Rust runtime + cargo-binstall + tools
```

## What's Installed

| File | Resources |
|------|-----------|
| `tools.cue` | ripgrep, fd, jq, bat (via aqua registry) |
| `runtime.cue` | Go runtime + gopls, staticcheck, goimports (via `go install`) |
| `rust.cue` | Rust runtime (rustup) + cargo-binstall + binstall installer + eza, hyperfine, tokei |

## Usage

```bash
tomei apply examples/minimal/
```

## Notes

- `cue.mod/` and `tomei_platform.cue` are generated by `tomei cue init` and included in this example.
- All resources are defined without `import` — useful for understanding the raw schema.
- See [real-world](../real-world/) for the same patterns using presets.
